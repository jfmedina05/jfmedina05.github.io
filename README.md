## How to View Locally

To view the website locally:

```bash
git clone https://github.com/jfmedina05/jfmedina05.github.io.git
cd jfmedina05.github.io
```

Then open:

```text
index.html
```

in a web browser.

You can also use a simple local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

---

## Deployment

This website is deployed using GitHub Pages.

Because this repository is named:

```text
jfmedina05.github.io
```

GitHub Pages automatically serves the site at:

```text
https://jfmedina05.github.io/
```

Any updates pushed to the `main` branch will update the live website after GitHub Pages rebuilds the site.
